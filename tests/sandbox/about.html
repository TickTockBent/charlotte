About Charlotte

Charlotte is an MCP server that renders web pages into structured, agent-readable representations using headless Chromium and Puppeteer.

It exposes the browser's semantic understanding — the accessibility tree, layout geometry, and interactive elements — to AI agents via the Model Context Protocol.

This allows agents to navigate, observe, and interact with web pages without needing vision models or brittle CSS selectors.
